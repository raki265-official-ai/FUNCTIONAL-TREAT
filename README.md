# FUNCTIONAL-TREAT PROJECT

📌 Project Name

Data Analyzer and Transformer

📖 About Project

This Python project is made to perform different operations on 1D and 2D lists using functions.

The program uses:

1 -Built-in Functions

2-User-defined Functions

3-Lambda Functions

4-Recursion

5-*args and **kwargs

6-Global Variable

7-Sorting

8-Multiple Return Values

It works using a simple menu-driven interface.

🎯 Objective

The main aim of this project is to analyze and transform list data in an easy way using Python functions.

🛠 Features

✔ Find sum, minimum, maximum, and average

✔ Display unique and duplicate values

✔ Use recursion for factorial

✔ Filter values using lambda

✔ Sort 1D and 2D lists

✔ Use *args and **kwargs

✔ Return multiple values from functions

✔ Display data in formatted style


📂 Concepts Used

1. Built-in Functions

Functions like:

◾len()

◾sum()

◾min()

◾max()


are used to analyze data.

2. User-defined Functions

Custom functions are created for:

◾Average calculation

◾Duplicate checking

Unique values

◾Sorting

3.  *args and **kwargs4. 
*args → accepts multiple values

**kwargs → displays key-value information

4. Recursion

A recursive factorial function is used.

Example:
6! = 6* 5 × 4 × 3 × 2 × 1

5. Lambda Function

Lambda is used with:

filter()
map()

for data transformation.

6. Global Variable

A global summary variable stores:

Total values

Overall average

7. Sorting
   
◾sort() for 1D list

◾sorted() for 2D list

▶ How to Run

Open Python or jupyter notebook

Save the file as:.ipynb file
and run the program

✅ Conclusion

This project helps in understanding Python functions and list operations in a simple and practical way. It improves knowledge of functional programming concepts using real examples.
